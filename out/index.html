How it works



- Aim is a python package.
- Use it to track all your ML metadata - hyperparams, metrics, media, distribution, etc.
- Only two functions to integrate with your training code.
- Works with any python script, notebook and ML framework.
- Stores metadata logs locally and remotely.
- Comes with the most powerful experiment comparison UI.

pip3 install aim
from aim import Run
                    
run = Run()

# Save inputs, hparams or any other 'key: value' pairs
run['hparams'] = hyperparam_dict

for step in range(10):
  # Log metrics to visualize performance

Features:

Track and search

Log all important metadata into Aim and easily compare every aspect of your runs— metrics, hyperparameters, dependencies, system metrics, and more—to understand differences in model performance.

Visualize Anything

Aim's performant UI allows you to record, transform, compare and visualize all metadata giving you full control over the kind of comparisons you need to best visualize your runs.

Log all your AI metadata

Images, audio files, scatter plots. Log everything you need for the granular analyses of your runs.
